Implementing your own fittings¶

To implement your own fittings simply create a class that extends the Fitting class (see below). Then place the class in one of your configured fittings directories.

class morgoth.fittings.fitting.Fitting[source]¶

Fittings are plugins to morgoth that expose behaviors

Fittings can be used to insert data into morgoth via any format needed. Or they can be used to retrieve the data from morgoth in any desired format.

This class is the base class for all Fittings
