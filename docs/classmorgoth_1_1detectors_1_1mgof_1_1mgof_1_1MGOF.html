Create an anomaly detector. 

     This anomaly detector uses a multinomial goodness-of-fit test.

     The algorithm is adapted from this <a href='http://www.hpl.hp.com/techreports/2011/HPL-2011-8.html'>paper</a>
Parameters
windows,:list of (timedelta, timedelta) tuples The first element is the offset of the window The second element is the duration of the window
period,:timedelta object of how much time to wait between checking windows for anomalies
duration,:the length of the window to analyze, timedelta object
n_bins,:the number of discrete values to use in analyzing the metric data
normal_count,:the number of windows of a given pattern that must be found in order to count that pattern as normal
chi2_percentage,:a value between 0-1 used to determine the statistical probalility of a window matching a given pattern Increasing this percentage will make the algorithm less tolerant of differences between windows.
